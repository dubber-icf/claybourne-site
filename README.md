# Claybourne

Website for the relaunch of *Claybourne*, the award-winning 1998 New Zealand radio drama, as a podcast.

## About

An American corporate troubleshooter arrives in a remote coastal town in the far north of New Zealand and finds a body on the beach, a government conspiracy, a sentient AI, an ancient Māori prophecy, and a Taniwha. Ninety-six episodes. Twin Peaks meets Stranger Things in late 90s Aotearoa.

The original radio series was produced by Pronoun Productions (Andrew Dubber & Belinda Todd) in 1998 for Newstalk ZB, with the assistance of NZ On Air. It won Best Dramatic Production at the 1999 New Zealand Radio Awards.

## This repository

A static site built in plain HTML and CSS, using Cormorant Garamond (Google Fonts) and atmospheric imagery drawn from the series bible. Deployed to Vercel.

- `index.html` — landing page: hero, story, characters, Taniwha and Delilah feature panels, listen, subscribe
- `cast.html` — cast and crew, with a short historical note
- `images/` — character portraits, mood artwork, and background images

## Local preview

Open `index.html` in a browser. No build step.
